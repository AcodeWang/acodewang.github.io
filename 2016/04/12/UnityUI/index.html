UnityUI

##The New UI

先设置UI namespace

Anchor 是对于父物体而言的，表示子UI物体距离父物体锚点的位置。
Rect 到 Anchor点的距离是固定的。
Sprite 把图象分成9部分，可以用来拉伸。
ScrollRect 可以包括Rect比他大的子物体，用Mask遮挡其他部分。可以直接设置ScrollBar。
RenderOrder 在hierarchy层级菜单中从上到下渲染，先渲染的在后面（底层）。

##Using UI Tools

对于2D和UI 把Texture改为Sprite
Preserve Aspect 使Image保持最小，不随父物体Rect改变
使用多个Canvas做各种菜单，用Panel固定菜单大小。
按住Shift 改变Anchor的同时使UI组件Rect随Anchor改变。
UI交互时，可以选择Static或Dynamic 可以把Slider的值作为参数。
在不同的场景使用同一物体或UI时，有两种方法：

  1. 使用Prefab，读取新场景时将Prefab实例化载入。
  2. 可以在Scenes间传递，在Load时不DontDestroyOnLoad()。
